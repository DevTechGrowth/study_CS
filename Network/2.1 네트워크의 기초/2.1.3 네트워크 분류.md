## LAN
- Local Area Netwrok
- 근거리 통신망
- 규모 : 개인 소유 가능한 규모의 좁은 공간
- ex) 사무실, 건물, 캠퍼스 등
- 특징 : 전송 속도가 빠르고 혼잡하지 않음

## MAN
- Metropolitan Area Network
- 대도시 지역 네트워크
- 규모 : 도시 규모의 넓은 지역
- ex) 서울시
- 특징 : 전송 속도는 평균, LAN보다 더 혼잡
 
## WAN
- Wide Area Network
- 광역 네트워크
- 규모 : 세계 규모
- ex) 국가, 대륙
- 특징 : 전송 속도가 낮고 MAN보다 더 혼잡
